# tpChatMode
チャットのモードが自動で何とかなります。

* ログイン後、最初のキャラ起動時、PTであればPTチャットになります。
	* PTで無ければギルドチャットになります
* PTに加入したとき、PTチャットになります。
	* IDに自動入場したとき、PTチャットになります
	* PT解散時、PTチャットモードだと、ギルドチャットになります。
		* PT未所属で自動IDを抜けたときも同様です。
* キャラ/エリア/CH切り替え時、シャウトモードだと、PTであればPTチャットになります。
	* PTで無ければギルドチャットになります
	* シャウトモード以外では変更しません


##その他
* ギルド未加入を考慮してません
